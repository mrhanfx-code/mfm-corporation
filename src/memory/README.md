# Memory System

File-based memory system for MFM Corporation bot.

## Purpose
Persistent memory storage for user interactions, feedback, and project context.

## Structure
- `file-based-memory.js` - Core memory storage and retrieval
- `memory-types.js` - Memory categorization (user, feedback, project)
- `memory-persistence.js` - Save/load operations
- `memory-retrieval.js` - Context-based retrieval
- `memory-summarizer.js` - Long-term summarization

## Integration
Connected to agent-base.js for seamless memory operations.
