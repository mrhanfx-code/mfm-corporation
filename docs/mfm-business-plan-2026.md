# MFM Corporation — Business Plan 2026

*Generated: May 24, 2026 | Planning Horizon: 1-Year Operational + 3-Year Growth | Currency: MYR*

## Executive Summary

MFM Corporation operates a **zero-cost, revenue-first bootstrapped model**. All infrastructure runs on free tiers (Cloudflare Workers, OpenRouter free models, Supabase free tier). The business generates cash through **project-based AI automation and software development services**, then reinvests profits into building a SaaS product (reusable agent templates) and a social media client arm. Year 1 target: **MYR 60K–120K revenue**, break-even by month 10–12. Year 3 target: **MYR 500K–1M**, SaaS-led with 100+ customers. Malaysia Digital Acceleration Grant (MDAG) provides up to **70% co-funding (RM 2M cap)** for qualifying growth activities.

## 5. Zero Personal Capital Strategy

### 5.1 Free-Tier Infrastructure Stack

| Layer | Service | Cost | Limit |
|-------|---------|------|-------|
| Frontend | GitHub Pages | Free | 1GB bandwidth, 100GB storage |
| Backend | Cloudflare Workers | Free | 100,000 requests/day |
| Database | Cloudflare D1 | Free | 500MB storage |
| Cache/State | Cloudflare KV | Free | 1GB storage |
| File Storage | Cloudflare R2 | Free | 10GB/month |
| LLM API | OpenRouter (free models) | Free | Rate limited |
| LLM Fallback | Cloudflare Workers AI | Free | 10,000 requests/day |
| Email | SendGrid (free tier) | Free | 100 emails/day |
| Database (alt) | Supabase | Free | 500MB |
| CRM | HubSpot/Notion | Free | Basic features |
| Social Scheduling | Buffer | Free | 3 channels |
| Design | Canva | Free | Basic features |

### 5.2 API Cost Threshold & Scaling Plan

| Tier | Trigger | Action | Revenue Required |
|------|---------|--------|------------------|
| **Current** | All usage within free limits | Maintain status quo | MYR 0 |
| **Threshold 1** | OpenRouter free rate limits hit (e.g., 200 requests/day) | Implement caching, request batching, model fallback chain | MYR 2,000+/mo |
| **Threshold 2** | Cloudflare Workers free limit (100K/day) | Upgrade to Workers Paid ($5/mo) or optimize code | MYR 5,000+/mo |
| **Threshold 3** | Need paid LLM APIs for reliability | Cerebras ($0.60/M tokens) or OpenRouter paid tier | MYR 10,000+/mo |
| **Threshold 4** | Enterprise client demand | Dedicated paid infrastructure | MYR 20,000+/mo |

**Mitigation Strategies:**
1. **Model fallback chain:** Free OpenRouter → Cerebras (cheap) → Cloudflare Workers AI (free) → OpenRouter paid (last resort)
2. **Aggressive caching:** Cache common agent responses in KV for 1–24 hours
3. **Request batching:** Batch non-urgent tasks into single API calls
4. **Client rate limiting:** Each client gets quota; overage triggers upgrade conversation
5. **Offline agent mode:** For simple tasks, use local rule-based logic without API calls
